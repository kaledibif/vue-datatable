# Flight Information DataTable

A basic DataTable Component for displaying Flight Information using a JSON file.

Features

- Initialized via Vue-CLI (VueJS 3.0)
- Simple Project Structure for better readability
- Flight info read from JSON file
- Ability of a multi-column filter
